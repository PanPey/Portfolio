Negative Behaviour Detecion

This was a project I started back in August 2020 when I first learned python and ML. I along with two of my friends created this project for our college semester.

Used Tweepy- a python library to fetch live tweets with the help of Twitter API and Textblob to analyze the sentiment of tweets.

Created a Negative Comment Classifier implementing NB-SVM and logistic regression models.

Finally did multi-class classification of the tweets into six categories toxic, severe toxic, obscene, threat, insult, and identity hate and graphically visualized the results using Seaborn andMatplolib libraries of python.

Technologies:
  • - Python
  • - Machine Learning
